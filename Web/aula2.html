Containers

Organizando conteúdo

  • header: cabeçalho da página
  • nav: navegação da página
  • section: seções da página
  • footer: rodapé
  • aside: conteúdo estra que fica ao lado
  • article: conteúdos independentes da página
  • div: container genérico

Atributos

  • id: identificador único
  • href: indica a localização de um recurso
  • src: indica a localização de uma mídia
  • alt: texto alternativo
  • class: especificar classes css
  • style: permite adicionar css
  • title: tool ip (texto quando o mouse fica em cima)
